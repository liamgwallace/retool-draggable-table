# Retool Draggable Table

Polished draggable table custom component for Retool.

## Local dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Retool sync

```bash
npm run retool:login
npm run retool:init
npm run retool:dev
```

Deploy a revision with:

```bash
npm run retool:deploy
```

## Export

The Retool component export is `RetoolDraggableTable` from `src/index.tsx`.
